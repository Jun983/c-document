# C 언어 입문 학습 자료

프로그래밍 지식이 없는 사람도 따라올 수 있도록 만든 **C 언어 학습 자료 저장소**입니다.  
문법을 빠르게 훑는 것보다, "왜 이렇게 쓰는지"를 이해하고 직접 실습해 보는 데 초점을 맞춥니다.

## 학습 대상

- 코딩을 처음 접하는 입문자
- C 언어를 처음 배우는 학생/취준생
- 문법 암기보다 예제 중심으로 기초를 다지고 싶은 학습자

## 학습 목표

- C 언어의 핵심 문법을 단계적으로 이해하기
- 컴파일/실행 과정에 익숙해지기
- 에러 메시지를 읽고 스스로 수정하는 습관 만들기
- 작은 문제를 C 코드로 해결하는 기본 문제 해결력 기르기

## 시작하기

개발 환경 설정이 필요하다면 아래 가이드를 먼저 확인하세요.

- [개발 환경 설정 가이드](docs/guide/development-environment-setup.md)

## 추천 학습 순서

1. 개발 환경 설정(컴파일/실행 확인)
2. C 기본 문법(변수, 자료형, 연산자, 조건문, 반복문)
3. 함수와 배열/문자열
4. 포인터와 메모리 개념
5. 파일 입출력과 간단한 프로젝트 실습

## 기여(선택)

오탈자 수정, 설명 보완, 입문자 관점의 예제 개선은 언제든 환영합니다.  
변경 시에는 "처음 배우는 사람이 이해하기 쉬운가?"를 기준으로 작성해 주세요.

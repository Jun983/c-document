# c-document

C 언어 학습·정리용 문서 프로젝트입니다. Obsidian 등 마크다운 기반 도구로 노트를 관리하는 것을 전제로 합니다.

## 목적

- C 문법, 표준 라이브러리, 자주 쓰는 패턴을 한곳에 모아 두기
- 예제 코드·빌드 산출물은 저장소에 넣지 않도록 `.gitignore`로 정리

## 시작하기

```bash
git clone <저장소 URL>
cd c-document
```

이후 선호하는 에디터(Obsidian, VS Code 등)로 폴더를 열어 문서를 편집하면 됩니다.

## 저장소에 포함·제외되는 것

- **포함**: 마크다운 문서, 이미지 등 노트 자료, Obsidian 플러그인 설정(`.gitignore`에서 제외하지 않은 범위)
- **제외**: Obsidian 로컬 UI 상태(`workspace.json` 등), 캐시, C/C++ 빌드 산출물(`.o`, `.exe`, `build/` 등) — 자세한 항목은 `.gitignore` 참고

## 문서 작성 시 참고

- 예제를 C 소스로 남기려면 `*.c` / `*.h` 파일로 두고, 컴파일 결과물은 커밋하지 않습니다.
- 팀·공개 저장소라면 개인 정보나 비밀 키가 노트에 들어가지 않도록 주의하세요.

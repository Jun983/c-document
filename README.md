# C 언어 입문 학습 자료

프로그래밍 지식이 없는 사람도 따라올 수 있도록 만든 **C 언어 학습 자료 저장소**입니다.  
문법을 빠르게 훑는 것보다, "왜 이렇게 쓰는지"를 이해하고 직접 실습해 보는 데 초점을 맞춥니다.

## 학습 대상

- 코딩을 처음 접하는 입문자
- C 언어를 처음 배우는 학생/취준생
- 문법 암기보다 예제 중심으로 기초를 다지고 싶은 학습자

## 학습 목표

- C 언어의 핵심 문법을 단계적으로 이해하기
- 컴파일/실행 과정에 익숙해지기
- 에러 메시지를 읽고 스스로 수정하는 습관 만들기
- 작은 문제를 C 코드로 해결하는 기본 문제 해결력 기르기

## 1. 변수란 무엇인가

먼저 **변수**를 정확히 이해해 봅시다.

- 변수는 값을 저장하기 위해 **메모리**에 확보한 공간에 붙인 **이름**입니다.
- 프로그램은 변수 이름을 사용해 메모리의 값을 **저장**하고 **읽습니다**.
- 변수는 선언할 때 **자료형**을 함께 지정하며, 자료형은 저장 가능한 값의 종류와 메모리 크기에 영향을 줍니다.

**메모리**는 프로그램이 실행되는 동안 데이터를 보관하는 공간입니다.

- 변수를 선언하면 해당 변수에 필요한 크기만큼 메모리가 **할당**됩니다.
- 각 변수는 메모리 안에서 **주소**를 가집니다.

예를 들어 아래 코드는 `age`라는 정수형 변수에 20을 저장하는 코드입니다.

```c
int age = 20;
```

`=`는 오른쪽 값을 왼쪽 변수에 저장하는 **대입 연산자**입니다.

## 2. C의 기본 자료형

C 언어에서는 저장할 값의 종류에 맞춰 자료형을 선택해야 합니다.
아래 4가지는 입문 단계에서 **자주 사용하는 자료형**입니다.

### 1. `int` (정수형)

- 소수점이 없는 정수를 저장합니다.
- 예: 나이, 개수, 점수
- 아래 예시는 정수형 변수 `age`를 선언하고, 값 `20`을 저장하는 코드입니다.

```c
int age = 20;
```

### 2. `float` (실수형, 단정도)

- 소수점이 있는 값을 저장합니다.
- `double`보다 정밀도는 낮지만 메모리를 덜 사용합니다.
- 아래 예시는 실수형 변수 `height`를 선언하고, 소수점 값 `175.5`를 저장하는 코드입니다.

```c
float height = 175.5f;
```

### 3. `double` (실수형, 배정도)

- 소수점 값을 더 정밀하게 저장합니다.
- 일반적으로 실수 계산에서는 `float`보다 `double`을 더 자주 사용합니다.
- 아래 예시는 배정도 실수형 변수 `pi`를 선언하고, 원주율에 가까운 값을 저장하는 코드입니다.

```c
double pi = 3.141592;
```

### 4. `char` (문자형)

- 문자 1개를 저장합니다.
- 작은따옴표(`'`)를 사용합니다.
- 아래 예시는 문자형 변수 `grade`를 선언하고, 문자 `'A'`를 저장하는 코드입니다.

```c
char grade = 'A';
```

아래 표는 C에서 사용하는 기본 자료형을 한 번에 정리한 내용입니다.

| 자료형 | 분류 | 일반적 크기(바이트) | 설명 |
| --- | --- | --- | --- |
| `char` | 문자/정수 | 1 | 문자 1개 또는 작은 정수 값을 저장합니다. |
| `signed char` | 정수 | 1 | 부호가 있는 1바이트 정수입니다. |
| `unsigned char` | 정수 | 1 | 0 이상만 저장하는 1바이트 정수입니다. |
| `short` (`short int`) | 정수 | 2 | `int`보다 작은 범위의 정수입니다. |
| `unsigned short` | 정수 | 2 | 0 이상만 저장하는 `short` 정수입니다. |
| `int` | 정수 | 4 | 가장 기본적으로 사용하는 정수형입니다. |
| `unsigned int` | 정수 | 4 | 0 이상만 저장하는 `int` 정수입니다. |
| `long` (`long int`) | 정수 | 4 또는 8 | `int` 이상 범위의 정수입니다. |
| `unsigned long` | 정수 | 4 또는 8 | 0 이상만 저장하는 `long` 정수입니다. |
| `long long` | 정수 | 8 | 매우 큰 정수 값을 저장합니다. |
| `unsigned long long` | 정수 | 8 | 0 이상만 저장하는 큰 정수형입니다. |
| `float` | 실수 | 4 | 단정도 실수형입니다. |
| `double` | 실수 | 8 | 배정도 실수형입니다. |
| `long double` | 실수 | 8, 12 또는 16 | `double`보다 더 넓은 정밀도를 제공할 수 있습니다. |
| `_Bool` (`bool`) | 논리 | 1 | 참/거짓 값을 저장합니다. (`stdbool.h` 필요) |
| `void` | 특수 | - | 값이 없음을 나타내는 형식입니다. |

자료형의 정확한 크기는 컴파일러와 운영체제에 따라 달라질 수 있습니다.

### 실습

`variables.c` 파일을 만들고 아래 코드를 입력하세요.

```c
#include <stdio.h>

int main(void) {
    int age = 20;
    float height = 175.5f;
    double weight = 68.25;
    char grade = 'A';

    printf("나이: %d\n", age);
    printf("키: %.1f\n", height);
    printf("몸무게: %.2lf\n", weight);
    printf("등급: %c\n", grade);

    return 0;
}
```

정상이라면 아래처럼 출력됩니다.

```text
나이: 20
키: 175.5
몸무게: 68.25
등급: A
```

## 3. 상수와 리터럴

프로그램에서는 변하지 않는 값도 자주 사용합니다. 이때 **상수**와 **리터럴**을 구분해서 이해하면 좋습니다.

### 1. 리터럴(Literal)

- 리터럴은 코드에 직접 적는 고정된 값입니다.
- 예: `10`, `3.14`, `'A'`, `"Hello"`
- 아래 코드에서 `20`, `175.5f`, `'A'`는 모두 리터럴입니다.

```c
int age = 20;
float height = 175.5f;
char grade = 'A';
```

### 2. 상수(Constant)

- 상수는 값을 한 번 정하면 이후에 변경하지 않도록 만든 식별자입니다.
- C에서는 보통 `const` 키워드를 사용해 상수를 선언합니다.
- 상수 이름은 의미를 분명히 하기 위해 대문자 스타일을 자주 사용합니다.

```c
const double PI = 3.141592;
const int MAX_STUDENT = 30;
```

- 상수는 선언 후 값을 바꾸려고 하면 컴파일 오류가 발생합니다.

### 실습

아래 코드를 `constant_literal.c` 파일에 입력하세요.

```c
#include <stdio.h>

int main(void) {
    const int MAX_STUDENT = 30;    // 상수
    int current_student = 18;      // 리터럴 18 사용
    char grade = 'A';              // 리터럴 'A' 사용

    printf("최대 인원: %d명\n", MAX_STUDENT);
    printf("현재 인원: %d명\n", current_student);
    printf("평가 등급: %c\n", grade);

    return 0;
}
```

정상이라면 아래처럼 출력됩니다.

```text
최대 인원: 30명
현재 인원: 18명
평가 등급: A
```

## 4. 형변환

형변환은 어떤 자료형의 값을 다른 자료형으로 바꾸는 것을 의미합니다.

### 1. 자동 형변환 (Implicit Conversion)

- 컴파일러가 문맥에 맞게 자료형을 자동으로 바꾸는 방식입니다.
- 예를 들어 `int` 값을 `double` 변수에 저장하면 자동으로 `double` 형식으로 변환됩니다.
- 자동 형변환은 값을 더 큰 범위의 자료형으로 자연스럽게 옮길 때 주로 사용됩니다.

```c
int score = 90;
double avg = score;
```

### 2. 명시적 형변환 (Explicit Conversion)

- 개발자가 직접 원하는 자료형으로 변환하는 방식입니다.
- `(자료형)값` 형태로 작성합니다.
- 명시적 형변환은 소수점을 버리거나, 특정 자료형으로 결과를 맞춰야 할 때 직접 사용합니다.

```c
double height = 175.9;
int height_int = (int)height;
```

- 위 코드에서 `(int)`를 사용하면 소수점 아래 값은 버려지고 정수 부분만 남습니다.

### 실습

아래 코드를 `type_casting.c` 파일에 입력하세요.

```c
#include <stdio.h>

int main(void) {
    int score = 95;
    double score_double = score;      // 자동 형변환

    double height = 175.9;
    int height_int = (int)height;     // 명시적 형변환

    printf("정수 점수: %d\n", score);
    printf("실수 점수: %.1lf\n", score_double);
    printf("원래 키: %.1lf\n", height);
    printf("변환된 키: %d\n", height_int);

    return 0;
}
```

정상이라면 아래처럼 출력됩니다.

```text
정수 점수: 95
실수 점수: 95.0
원래 키: 175.9
변환된 키: 175
```
